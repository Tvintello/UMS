# Русский

## Гайд по установке
Чтобы установить данное расширение в качестве временного скачай zip файл, распакую его и следуй инструкциям для твоего браузера
### Для chrome
Вбей в адресную строку `chrome://extensions/`, включи `режим разработчика`, нажми на `Загрузить распакованное расширение`, найди папку с кодом
### Для firefox
Вбей в адресную строку `about:debugging` -> `Этот firefox` -> `Загрузить временное расширение` и найди `manifest_firefox.json` в папке с кодом


# English
## Installation guide
To install UMS as a temporary extension on your browser `download zip file with code` and unpack it then follow further instructions:
### For chrome:
Go to `chrome://extensions/` page then enable `developer mode`, click on `Load unpacked` then choose the folder with code
### For firefox:
Go to `about:debugging` page -> `This firefox` -> `Load temporary extension` and locate `manifest_firefox.json` file in the folder with code
